const std::string &  alias

inlinestatic

Loads content from the disk.

This function calls the template functions vlk::ConstructContent(const std::string) and vlk::DestroyContent(T*) internally. These functions must be specialized before they can be invoked.

This function will also load metadata from a corresponding meta file if one is found. See the guide on metadata files for information on how to write one.

This function sends a LoadEvent if the content is loaded successfully.

Parameters
pathThe path to the file to load the content from, relative to the current content prefix.
aliasThe alias to assign the content once loaded. This string will be used as the key to retrieve the content with vlk::Content<T>::GetContent(const std::string&).
Returns
true if the content loads successfully, false otherwise.
Thread Safety
Must only be called from the main thread.
Resource locking is handled internally.
Unique access to this class is required.
This function may block the calling thread.

The content prefix is a string that is prepended to the path argument of vlk::Content<T>::LoadContent(const std::string&, const std::string&) when loading content from the disk. By default, the content prefix is "content/".
