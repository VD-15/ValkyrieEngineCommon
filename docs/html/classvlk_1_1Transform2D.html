Detailed Description

2D geometric transform.

Combines a scale, rotation and translation into a single operation.

Sets the parent of this transform.

Parameters
trThe new parent of this transform or nullptr to remove the current parent. Additionally, this must not be equal to tr or any parent of tr.
Template Parameters
SSpecifies the behaviour when changing parent objects.
If Space::Local, this transform will not retain it's position in world space.
If Space::Global, this transform will attempt to retain the same position in world space.
Exceptions
