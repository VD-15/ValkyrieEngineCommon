Loads content from the disk. 

This function calls the template functions {\ttfamily vlk\+::\+Construct\+Content(const std\+::string)} and {\ttfamily vlk\+::\+Destroy\+Content(\+T$\ast$)} internally. These functions must be specialized before they can be invoked.

This function will also load metadata from a corresponding meta file if one is found. See the guide on metadata files for information on how to write one.

This function sends a {\ttfamily \mbox{\hyperlink{structvlk_1_1Content_1_1LoadEvent}{Load\+Event}}} if the content is loaded successfully.
